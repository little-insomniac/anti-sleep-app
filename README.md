### Option 2: From source

Requires [Node.js](https://nodejs.org) and [Git](https://git-scm.com).

```bash
git clone https://github.com/little-insomniac/anti-sleep-app.git
cd anti-sleep-app
npm install
npm start
```

## ✨ Features

- Stop your computer from going to sleep automatically
- No configuration required, just launch and go
- Minimal resource usage with system tray integration
